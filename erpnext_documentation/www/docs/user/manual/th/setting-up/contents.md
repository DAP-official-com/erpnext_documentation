# การตั้งค่า

การตั้งค่าระบบ ERP เปรียบเสมือนการเริ่มต้นธุรกิจของคุณใหม่ทั้งหมด แต่ในโลกเสมือนจริง โชคดีที่มันไม่ได้ยากเท่าธุรกิจจริง และคุณจะได้ทดลองด้วย!

เพื่อให้ได้สิทธิ์ในการใช้งาน ERP ผู้ดำเนินการต้องจัดสรรเวลาและดำเนินการใช้งานด้วยความทุ่มเท โดยปกติแล้ว นี่ไม่ใช่โปรเจ็กต์หลังเลิกงานที่คุณใช้เวลาสองสามชั่วโมง

คุณจะพบตัวเลือกการกำหนดค่าส่วนกลางที่นี่ สำหรับการตั้งค่าแบบโมดูล โปรดไปที่โมดูลที่เกี่ยวข้อง
