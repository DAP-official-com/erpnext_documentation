# การจัดการภาษี

**ภาษีเป็นเงินสมทบภาคบังคับสำหรับรายได้ของรัฐ/ประเทศ**

หนึ่งในแรงจูงใจหลักสำหรับการใช้เครื่องมือบัญชีภาคบังคับคือการคำนวณภาษี ERPNext ช่วยให้คุณสร้างเทมเพลตภาษีที่กำหนดค่าได้ซึ่งคุณสามารถนำไปใช้กับธุรกรรมการขายหรือการซื้อของคุณได้


## 1. บัญชีภาษี 

สำหรับบัญชีภาษีที่คุณต้องการใช้ในเทมเพลตภาษี ไปที่:
> หน้าหลัก > บัญชี > ผังบัญชี

เลือกบัญชีและคลิกแก้ไข เลือก 'ประเภทบัญชี' เป็น 'ภาษี' สำหรับบัญชี


## 2. เทมเพลตภาษีขายและค่าธรรมเนียม
ภาษีขายและค่าใช้จ่ายภาษีเทมเพลทลึกซึ้งสำหรับการทำธุรกรรมการขายของคุณเช่น[ใบสั่งซื้อ](/docs/user/manual/en/selling/sales-order) และ [ใบแจ้งหนี้](/docs/user/manual/en/accounts/sales-invoice).
อ่าน [เทมเพลตภาษีขายและค่าธรรมเนียม](/docs/user/manual/en/selling/sales-taxes-and-charges-template) เพื่อทราบข้อมูลเพิ่มเติม


## 3. เทมเพลตภาษีซื้อและค่าธรรมเนียม 
ภาษีซื้อและค่าใช้จ่ายภาษีเทมเพลทลึกซึ้งสำหรับการทำธุรกรรมการขายของคุณเช่น [ใบสั่งซื้อ](/docs/user/manual/en/buying/purchase-order) และ [ใบแจ้งหนี้](/docs/user/manual/en/accounts/purchase-invoice).
อ่าน [เทมเพลตภาษีขายและค่าธรรมเนียม](/docs/user/manual/en/selling/sales-taxes-and-charges-template) เพื่อทราบข้อมูลเพิ่มเติม

## 4. เทมเพลตภาษีสินค้า
ภาษีที่กำหนดในเท็มเพลตภาษีสินค้าใช้กับ [สินค้า](/docs/user/manual/en/stock/item) หรือ [กลุ่มสินค้า](/docs/user/manual/en/stock/item-group) เพื่อจะได้รับการตั้งค่ามากกว่าเทมเพลตภาษีการขาย/การซื้อ 
อ่าน [เทมเพลตภาษีสินค้า](/docs/user/manual/en/accounts/item-tax-template) เพื่อทราบข้อมูลเพิ่มเติม

## 5. หมวดหมู่ภาษี
หมวดหมู่ภาษีช่วยในการนำเทมเพลตภาษีการขาย/การซื้อไปใช้ในธุรกรรมของคุณโดยอัตโนมัติตามลูกค้า/ซัพพลายเออร์ที่เลือก
อ่าน [หมวดหมู่ภาษี](/docs/user/manual/en/accounts/tax-category) เพื่อทราบข้อมูลเพิ่มเติม
