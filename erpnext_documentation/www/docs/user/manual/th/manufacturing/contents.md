# การผลิต

โมดูลการผลิตใน ERPNext ครอบคลุมคุณลักษณะที่จำเป็นทั้งหมดที่ธุรกิจการผลิตต้องการ มีตัวเลือกในการสั่งงาน ป้อนสินค้าคงคลัง สร้างแผนการผลิต สร้างรายการหยุดทำงาน จัดการรายการวัสดุสำหรับสินค้าสำเร็จรูป / รายการเทมเพลต สร้างรายงาน และอื่นๆ

การผลิตประเภทต่างๆ เช่น การผลิตในสต็อก การสั่งทำ และวิศวกรตามสั่ง สามารถจัดการได้ในโมดูลการผลิตของ ERPNext
